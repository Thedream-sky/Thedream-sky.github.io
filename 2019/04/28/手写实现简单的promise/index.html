发表于 2019-04-28 | 分类于 前端

  • 重写promise 的resolve、reject 方法、then方法（重难点）（catch与之类似）

  • 几个关键变量：that保存this、state保存当前promise状态值、value变量用于保存 resolve 或者 reject 中传入的值、resolvedCallbacks与rejectedCallbacks分别保存 then 中的回调
